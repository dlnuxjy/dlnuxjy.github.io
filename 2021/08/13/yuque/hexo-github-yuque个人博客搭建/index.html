想拥有一个属于自己的博客吗？整理并记录日常工作生活中的点滴，还可以分享给世界各地的朋友，让更多的人认识和了解你。未完待续吧，总之懂得自然懂。

hexo

hexo 是一个快速，简介且高效的博客框架，更重要的是居然是免费的。使用 hexo 可以迅速建立属于你的博客，可以是本地的，也可以是网络的。hexo 中有很多的主题和插件可以选择，https://hexo.io/zh-cn/ 这里是官网。
