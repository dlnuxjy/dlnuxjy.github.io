vscode 安装

  1. vscode 官网下载 https://code.visualstudio.com/ window 版本，无脑 next 安装
  2. 插件安装 CMake 和 CMakeTool

cygwin 安装

  1. cygwin 官网下载 http://www.cygwin.com/  ** 找到 **setup-x86_64.exe，双击运行
  2. 选择Install from Internet ，然后下一页
  3. 选择安装路径 如 C:/cygwin64, Install for All Users,然后下一页
  4. 选择一个本地路径用于暂存软件包，安装完成后可删除，然后下一页，然后下一页
  5. 选择 https://mirrors.aliyun.com,然后下一页
  6. 安装 make gdb gdb-debuginfo gcc-g++ gcc-core gcc-debuginfo gcc-objc gcc-objc++ cygwin32-gcc-core cygwin32-gcc-debuginfo cygwin32-gcc-fortran cygwin32-gcc-g++

cmake 安装

  1. cmake 官网下载 https://cmake.org/download/ ，本文使用的是 cmake-3.21.0-rc3-windows-x86_64.msi
  2. 安装过程中注意选择增加环境变量 PATH

验证开发环境

配置 cmake 工程，编译运行

  1. 新建空文件夹 hello，注意不要有中文路径
  2. 使用 vscode 打开文件夹
  3. Ctrl+Shift+P快捷键，选择CMake: Quick Start，Scan for kit后，选择GCC X86-64-pc-cygwin
  4. 输入项目可执行程序名字，如hello,回车，选择Executable,等待配置完成
  5. 点击下 启动，当发现Termeinal中输出Hello,World字样，代表程序编译运行成功。

调试运行

  1. main.cpp 中加入断点
  2. 点击 ，启动调试运行，当发现自动跳转到 main.cpp 中的断点处，代表调试运行成功。
